  # Введение
  
  **Назначение продукта:** создание удобной и функциональной платформы для доставки еды, предоставление людям нового `уникального продукта` с обновленным современным функционалом
  
  **Функциональность:** Выбор продуктов, оформление доставки, отслеживание местоположения курьера, заказ продуктов на определенный срок

  # Примеры использования
  1. При нажатии на кнопку *адрес* можно изменить существующий или добавить новый
  2. Можно выбрать доставку из определенного магазина, при нажатии иконки *Ашан* будет доступна доставка только из данной сети магазинов
  3. При нажатии на кнопку *корзина* можно посмотреть продукты которые выбрали
  4. При нажатии иконки *профиль* можно изменить свое имя, дату рождения а также посмотреть доступные специальные предложения
  5. При нажатии кнопки *промокод* при заказе можно использовать купоны для скидки
  6. Можно выбрать любой из разделов продуктов для заказа, например при нажатии иконки *бакалея* к выбору будут доступны продукты только из этого раздела
  7. При нажатии кнопки *где курьер* можно увидеть местоположние курьера
  8. При нажатии иконки *ваш заказ* откроется вся нужная информация связанная с заказом
  9. При нажатии кнопки *отмена заказа* заказ будет отменен
  10. При нажатии кнопки *удалить аккаунт* можно удалить свой аккаунт навсегда
